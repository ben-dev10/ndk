Welcome to the **ndk** (nard development kit) docs!

## What is ndk?

It is a small open-source library of UI components, styles (themes), react hooks, TailwindCSS plugins, icons and several mini utilities (like `cn` and the likes). If you
are wondering what "nard" means, it is the last for 4 characters of my first name, "BerNARD".

The toolkit for now consists of 2 packages - `@_ndk/ui` and `@_ndk/motion`, see <Link href="/docs/ui/installation" className="text-fd-primary">installation guides</Link>.
If you'd like to know the rationale behind this project, see <Link href="/docs/ui/why-ndk" className="text-fd-primary">Why ndk?</Link>.

These packages basically build on top of (and extends) many amazing open-source frontend libraries and frameworks including Next.js, Shadcn UI, TailwindCSS, Lucide-react and many more.
